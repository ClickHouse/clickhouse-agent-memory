Agents Architecture Style guide
Agent catalogue

Four AI agents, one ClickHouse cluster. Pick an agent, ask a sample question, or open the session report to see exactly how it works.

Every agent runs on the same three memory tiers (HOT event streams, WARM semantic search, GRAPH relationships) and the same eight MCP tools. What changes is the system prompt and the canonical question shape. Every session generates a precision report you can share with a solution architect or an enterprise buyer.

HOT tier

Live event streams on the ClickHouse Memory engine. Sub-5ms reads, volatile, perfect for "what is happening right now" questions.

WARM tier

Historical records with HNSW vector similarity on MergeTree. 50-500ms, persistent. Answers "have we seen this before" via semantic search.

GRAPH tier

Multi-hop traversal via SQL self-JOIN + UNION ALL on MergeTree edge tables. 3-10 ms at 2 hops. Maps blast radius, topology impact, and lateral-movement risk.
